Summary Report Help

The Summary Report presents an aggregate view of the selected block groups: there is one record for each summary metric. The source of the data in this tab is the Census 2010.

By clicking on a row you can highlight the individual row.

Clicking the Use Comparison box will bring up a drop down list showing values that can be used to compare with selected data.

The Summary Report can be exported to an Excel or CVS data file using the Export Results selector and button.

For the Council of Government / Metropolitan Planning Organization layer, the ACS 2014 data was interpolated down to census blocks and allocated to the appropriate geometry using a winner takes all method where the block data would go to the geometry in which the centroid of the block was contained. These numbers were rounded and may not always add up exactly to 100% as expected.
