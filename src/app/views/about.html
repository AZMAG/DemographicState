About the Statewide Demographic Viewer
The Statewide Demographic Maps Viewer allows you to graphically view selected population and housing data for all or virtually any block group subset of the state of Arizona. The viewer provides interactive mapping and reporting tools to allow exploration of data from Census 2010 and the American Community Survey (ACS) 5-year 2012-2016. The data is shown by census block group geography. If you have any comments, please call MAG at (602) 254-6300 or visit our web site at www.azmag.gov.
