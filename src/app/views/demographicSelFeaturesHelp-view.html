Selected Features Help

The Selected Features presents an aggregate view of the selected block groups: there is one record for each summary metric. The results in this table use census 2010 block group data.

The number of selected features retuned by Custom Summary or Advanced Query is displayed at the top of the table.

Selected features can also be sorted by each column in accending and decending order by clicking on each column header.

Hovering over a row will highlight the Census Block Group it represents on the map.

By clicking on a row you can highlight the individual row.

The selected features can be exported to an Excel or CVS data file using the Export Results selector and button.
