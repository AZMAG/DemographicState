Legend Help

The legend describes the contents of the current map. The content of the map is organized into layers that group similar types of geographic features.

The map contains two types of layer contents thematic maps contents and layer contents. The thematic map contents are controlled by the Maps button and the layer contents are controlled by the Layers button.

    The map contains the following layers:
  • Thematic Map

    • The thematic map layer is the colored census blocks. The description of the current thematic map is displayed at the top of the legend followed by the classes contained within the map. Each class describes the shade on the map used to display a range of values.

  • Layer Options

    • The Layer options are different geographic areas that can be added as an overlay to the thematic map.

You can change the display of the map's content in the following ways:

  • Change the visibility of a layer by checking or un-checking the layer's checkbox.
  • Change the transparency of the census block group layer by moving the transparency slider. Moving the slider to the left will increase the layer's transparency which will allow for seeing the contents underneath the layer.
  • To change the current demographic map, choose an item from the maps menu.
